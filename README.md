# Vite + React + TypeScript + shadcn/ui

Frontend project built with **Vite**, **React 18**, **TypeScript**, **Tailwind CSS**, and **shadcn/ui**.

---

## Requirements

* Node.js **18.x or 20.x**
* npm / pnpm / yarn 

---

## Install

```bash
npm install
```

---

## Run (Development)

```bash
npm run dev
```

Access:

```
http://localhost:5173
```

---

## Build

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```

---

## Tests

```bash
npm run test
```

---

## Lint

```bash
npm run lint
```

---

## Environment Variables

Create a `.env` file if needed:

```env
VITE_API_BASE_URL=http://localhost:3000
```

---

## Notes

* This project contains **frontend only**.
